# 🎮 Blocks

**Blocks** é um jogo arcade minimalista desenvolvido na Godot Engine 4.3. O objetivo central da jogabilidade é desafiar o jogador a destruir blocos na tela para avançar e pontuar.

---

## 🚀 Funcionalidades

* **Mecânica Principal**: Destruição ágil de blocos interativos.
* **Desenvolvimento Atualizado**: Estruturado na versão estável do Godot 4.3.
* **Código Otimizado**: Scripts limpos escritos puramente em GDScript.

---

## 🛠️ Tecnologias

* **Engine**: Godot Engine 4.3
* **Linguagem**: GDScript


---

## 💻 Como Rodar o Projeto

Para testar ou modificar o código-fonte, você precisa ter a Godot Engine 4.3 instalada.

```bash
# Clone o repositório
\$ git clone https://github.com[Seu-Usuario]/Blocks.git

# Importe o projeto na Godot
# Selecione o arquivo project.godot
```
